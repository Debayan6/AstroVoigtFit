AstroVoigtFit documentation
===========================

--------------------
Introduction
--------------------

``AstroVoigtFit`` is a Python package for fitting Voigt profiles to astronomical data. It is designed to handle spectral data (EDIBLES survey). It has the capability 
to fit multiple clouds with multiple species with multiple line.
